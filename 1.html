HTML

Hypertext Markup Language(HYML) is the standard markup language for creating web pages and web applications. Web browers receive GTML documents from a web server or from loval storage and render them into multimedia web pages. HTML describes the structure of a web page semantically standardoriginally included cues for the appearance of the document.

HTML elements are the building blocks of HTML pages. With GTML contructs, images and other objects, such as interactive forms, may be embedded into the rendered page. It provides a means to create structured documents by denoting structural semantics for text for such as headings, paragrahs, lists, links, quetes and other items. HTML elements are delineated by tags, written using angle brakets.
